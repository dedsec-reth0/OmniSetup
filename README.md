# 🚀 OmniSetup

**OmniSetup** is a universal, cross-platform CLI tool designed for instant configuration, updates, and basic maintenance of **Windows, macOS, and Linux** systems right after a clean OS installation.

> **Key Feature:** Runs with a single command on any OS directly from GitHub. Zero pre-installation required!

---

## ⚡ Quick Start

Open your terminal (or PowerShell as Administrator on Windows) and paste the command below:

```bash
bash <(curl -fsSL https://github.com/dedsec-reth0/OmniSetup/releases/download/Release/omnisetup.cmd)

```powershell
iex (irm https://github.com/dedsec-reth0/OmniSetup/releases/download/Release/omnisetup.cmd)
